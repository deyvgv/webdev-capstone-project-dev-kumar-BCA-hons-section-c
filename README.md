# Shoppy – E-Commerce Capstone Project

Hi! This is my **E-Commerce Web Interface** built as the **Capstone Project** for Foundations of Web Development at K.R. Mangalam University.

This project demonstrates a real-world e-commerce frontend with filters, search, sorting, cart system, and a fully responsive layout.

---

## What’s Inside
- **Responsive UI Layout** with hero section and navigation  
- **Product Grid** with 12 items, ratings, prices, categories, and local images  
- **Filtering Features** (rating, price range, categories)  
- **Live Search** updates products instantly  
- **Sorting System** for price and rating  
- **Dark/Light Theme Toggle** with saved preference  
- **Shopping Cart System**  
  - Add, remove, update quantity  
  - Subtotal calculation  
  - LocalStorage persistence  
  - Slide-in cart drawer  
- **Contact Form** with clean responsive UI  
- Fully mobile-friendly

---

## Tech Used
- **HTML5** (Semantic layout)
- **CSS3** (Flexbox, Grid, Animations, Responsive design)
- **JavaScript (Vanilla JS)**  
  - Filters, search, sorting  
  - Theme toggle  
  - Shopping cart logic  
  - LocalStorage  
- **Local Product Images**
- **Google Fonts (Poppins)**

---

## How to Open
1. Keep the following files together:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `images/` (folder with all product PNGs)

2. Open **index.html** in any web browser.
